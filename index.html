Weather App

A weather app made from scratch, using vanila JS. A simple app that fetches data from an DarkSky API based on user's geolocation. This project uses the Module pattern.

Featured Project

CryptoTrack App

SPA that allows the user to track the values of cryptocurrencies and the value of the cryptocurrencies the user has. The data is shown in the table, one table for each currency. Every table contains the name of the cryptocurrency, short name, value,% of the change in the last 24h, a form that when the user submits calculates and saves in localStorage. Data is refreshed every 60 seconds. Application has implemented pagination.

  • React
  • Sass
  • Bootstrap
