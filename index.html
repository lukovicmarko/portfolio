Featured Project

CryptoTrack App

SPA that allows the user to track the values of cryptocurrencies and the value of the cryptocurrencies the user has. The data is shown in the table, one table for each currency. Every table contains the name of the cryptocurrency, short name, value,% of the change in the last 24%, a form that when the user submits calculates and saves in localStorage.Data is refreshed every 60 seconds. Application has implemented pagination.

  • Sass
  • Bootstrap
